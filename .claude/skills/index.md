# Skills Index (DocuRAG)

Skills are the canonical “how we work” layer for agents. Load the smallest set that covers the task.

| Skill | Covers |
|------|--------|
| `core-architecture` | Modulith boundaries, module ownership, dependency decisions |
| `domain-modeling` | domain concepts and invariants (documents/chunks/eval) |
| `repository-design` | JDBC + SQL + Flyway + pgvector conventions |
| `api-design` | REST + Thymeleaf edge, OpenAPI alignment, DTO evolution |
| `ai-provider` | Spring AI wiring, profiles, provider configuration |
| `testing` | unit/IT/E2E strategy, fixtures, Testcontainers/Playwright |
